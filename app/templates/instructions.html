¿Cómo usar el Foliador PDF?

Ir al Foliador
01

Carga tu PDF

Arrastra tu archivo PDF a la zona de carga o haz clic para seleccionarlo. El archivo puede pesar hasta 2 GB.

02

Configura el foliado

Define el número de folio inicial, el rango de páginas a foliar y la esquina donde aparecerá el número.

03

Verifica la vista previa

La vista previa muestra la primera página foliada. Ajusta la posición y tamaño hasta que quede correcto.

04

Descarga el resultado

Haz clic en "Foliar y Descargar PDF". El archivo se descarga automáticamente con el nombre Foliado_tunombre.pdf.

Notas importantes

  • El folio se formatea siempre a 4 dígitos - ejemplo: 1 - 0001
  • Puedes foliar solo un rango de páginas dejando el resto sin número
  • La vista previa tiene un límite de 30 MB para proteger el servidor
  • El PDF original no se modifica, siempre se genera un archivo nuevo
  • No se aceptan PDFs con contraseña
