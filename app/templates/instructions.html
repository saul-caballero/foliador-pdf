¿Cómo usar el Foliador PDF?

01

Carga uno o varios PDFs

Arrastra uno o varios archivos PDF a la zona de carga o haz clic para seleccionarlos. Cada archivo puede pesar hasta 2 GB cada uno individualmente. Si cargas varios, se descargarán juntos en un ZIP.

02

Organiza los archivos

Reordena los archivos arrastrándolos en la lista o usa los botones A-Z y Z-A. También puedes agregar más archivos o eliminar los que no necesites sin recargar la página.

03

Configura el foliado

Define el número de folio inicial, el rango de páginas a foliar y la esquina donde aparecerá el número. El rango estimado de folios se muestra automáticamente.

04

Verifica la vista previa

La vista previa muestra la primera página foliada de cada archivo. Usa las flechas del carousel para revisar cada documento antes de procesar.

05

Confirma y descarga

Al hacer clic en "Foliar y Descargar" se muestra un resumen con el número de archivos, páginas totales y rango de folios. Confirma para iniciar la descarga.

06

Folios consecutivos entre archivos

Al cargar varios archivos, los folios continúan automáticamente entre documentos. Si el primer PDF tiene 50 páginas, el segundo empieza en el folio 0051.

Notas importantes

  • El folio se formatea siempre a 4 dígitos - ejemplo: 1 a 0001
  • Puedes foliar solo un rango de páginas dejando el resto sin número
  • La vista previa tiene un límite de 30 MB para proteger el servidor
  • El PDF original no se modifica, siempre se genera un archivo nuevo
  • No se aceptan PDFs con contraseña
  • Los archivos con problemas se marcan con ⚠ en la lista antes de procesar
  • La configuración de estilo y posición se guarda automáticamente entre sesiones
  • Puedes cambiar entre modo oscuro y claro con el botón en el encabezado
  • PDFs escaneados con rotación de metadatos son soportados correctamente
